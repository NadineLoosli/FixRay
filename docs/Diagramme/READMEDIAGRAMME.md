# Klassendiagramm bearbeiten mit draw.io

## Was ist hier gespeichert?

- **Klassendiagramm.drawio.png**: Das aktuelle Klassendiagramm als Bild – wird direkt in GitHub angezeigt.
- **Klassendiagramm.drawio**: Die bearbeitbare Version für draw.io (diagrams.net).

## So bearbeitest du das Diagramm

1. **Datei herunterladen**  
   Lade die Datei `Unbekanntes Diagramm.drawio` aus diesem Ordner auf deinen Rechner herunter.

2. **Draw.io öffnen**  
   Gehe auf [https://app.diagrams.net/](https://app.diagrams.net/).

3. **Datei öffnen**  
   Wähle „Gerät“ (Device) als Speicherort und öffne die gerade heruntergeladene Datei.

4. **Diagramm bearbeiten und speichern**  
   Bearbeite das Diagramm wie gewünscht.  
   Speichere anschließend die `.drawio`-Datei für die Weiterbearbeitung.

5. **Als PNG exportieren**  
   Gehe in draw.io auf **Datei > Exportieren als > Bild (PNG)**  
   Exportiere das Bild und überschreibe (falls gewünscht) die vorhandene `Klassendiagramm.drawio.png`.

6. **Beide Dateien wieder ins Repository hochladen**  
   Lade die aktualisierte `.drawio` sowie das neue Bild `.drawio.png` in diesen Ordner und ersetze die alten Versionen.

---

**Hinweis:**  
- Die `.drawio`-Datei ist das Original – sie enthält das Diagramm als XML, welches in draw.io/diagrams.net geladen werden kann.
- Die `.png` ist nur zum Anschauen, nicht zum Editieren.
